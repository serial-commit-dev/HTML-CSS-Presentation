An Introduction to
HTML & CSS

World Wide Web (WWW)

  • Information System: The Web is a system of interconnected documents and resources.
  • Content Sharing: It enables easy sharing of documents, images, and applications over the Internet.
  • Protocol: Resources are accessed and transferred according to the rules of the Hypertext Transfer Protocol (HTTP).
  • Audience: Designed to be user-friendly, appealing to users far beyond IT specialists.

Hyper Text Markup Language (HTML)

  • Standard Markup Language: HTML is the foundational language for documents displayed in a web browser.
  • Defines Structure & Content: Its primary role is to define the content (text, images) and the structure (headings, paragraphs, lists) of web pages.
  • Assisted By: HTML works closely with:
    • Cascading Style Sheets (CSS): For visual presentation and layout.
    • Scripting Languages (e.g., JavaScript): For interactivity and dynamic features.

Cascading Style Sheets (CSS)

  • CSS is a style sheet language used for specifying the presentation and styling of a document.
  • It is designed to work with markup languages like HTML or XML.
  • CSS is considered a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Timothy John Berners-Lee

  • Sir Tim Berners-Lee is an English computer scientist best known as the inventor of the World Wide Web (WWW).
  • He is credited with inventing the fundamental technologies: HTML, the URL system, and HTTP.
  • He currently holds professorial research positions at the University of Oxford and is a professor emeritus at MIT.

Hakon Wium Lie

  • Hakon Wium Lie is a Norwegian web pioneer and standards activist.
  • He is best known for developing Cascading Style Sheets (CSS) in 1994 while working at CERN with Tim Berners-Lee.
  • He served as the Chief Technology Officer (CTO) of Opera Software from 1998 to 2016.
  • He is currently the chairman of YesLogic, which develops Prince CSS-based PDF rendering software.
